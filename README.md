# Password Generator

A simple Python GUI application to generate strong, random passwords. Built with **Tkinter**, it allows users to specify password length and generate passwords containing letters, numbers, and special characters.  

---

## Features

- Generate random passwords with customizable length
- Includes letters (uppercase & lowercase), numbers, and special characters
- Simple and easy-to-use GUI
- Immediate display of generated password
